CRM Integration Gantt

Simple static web page visualizing a CRM integration timeline using a Gantt-style layout.

🔍 Overview

This project is a lightweight HTML-based visualization of a CRM integration plan. It provides a clear timeline of phases, tasks, and dependencies without requiring any backend or complex tooling.

The goal was to create a fast, portable, and easily deployable solution for sharing project timelines.

🚀 Live Demo

👉 https://vojtovapipelina.github.io/crm_gantt/

🛠️ Tech Stack
HTML5
CSS3
JavaScript (vanilla)
GitHub Pages (hosting & deployment)
⚙️ How it works

The page is fully static and rendered directly in the browser. There is no backend or data source — all data is embedded in the HTML/JS.

Deployment is handled via GitHub Pages:

Push to main branch
Automatic build & publish
Available via public URL
📦 Deployment

This project uses GitHub Pages for hosting.

Steps:

Upload index.html to repository
Enable GitHub Pages in repository settings
Select main branch and root folder
GitHub automatically deploys the site
